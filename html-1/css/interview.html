CSS Flexbox Interview Questions & Answers

1.What is Flexbox and why it is used?

Answer

Flexbox is a one-dimensional layout method for arranging items in rows or columns

The purpose of flexbox — flexibly lay out a set of block or inline elements in one dimension.

Flex terminology — flex container, flex item, main axis, and cross axis.

2. What are the two main axes in Flexbox?

Answer

main-axis:The main axis is defined by flex-direction, which has four possible values:

  • rows
  • row-reverse
  • column
  • column-reverse

cross axis:The cross axis run perpendicular to the main axis

3. How do you make an element a flex container?

Answer

we have to use display properties to make an element a flex container either flexorinline-flex

4. Explain the difference between display: flex; and display: inline-flex;.

Answer

display:flex;→The flex container itself behaves as a block-level element, taking up the full available width and forcing a line break after it.

display:inline-flex;→The flex container itself behaves as an inline-level element, only taking up the necessary width and allowing other inline elements to sit alongside it.

5. What is flex-direction and what are its possible values?

Answer

The flex-direction CSS property sets how flex items are placed in the flex container defining the main axis and the direction (normal or reversed).

The possible values are:

  • row

    By default row will locate the flex items side by side

  • row-reverse

    To reverse the order of elements in CSS using flexbox

  • column

    Flex items are laid out in a column (top to bottom).

  • column-reverse

    Flex items are laid out in a column in reverse order (bottom to top).

6. What is justify-content and how does it work?

Answer

The justify-content property defines how flex items are distributed along the main axis of the flex container, after accounting for any free space.

Common values include:

  • flex-start (default): Items are packed towards the start of the main axis.
  • flex-end: Items are packed towards the end of the main axis.
  • center: Items are centered along the main axis.
  • space-between: Items are evenly distributed with the first item at the start and the last item at the end.
  • space-around: Items are evenly distributed with equal space around them.
  • space-evenly: Items are evenly distributed with equal space between them and equal space at the ends.

7. What is align-items and how does it work?

Answer

The align-items property defines how flex items are distributed along the cross axis of the flex container.

the values of align-items are:

  • stretch (default): Items are stretched to fill the container (but still respect min-width/max-width).
  • flex-start: Items are start toward the cross axis.
  • flex-end: Items are packed towards the end of the cross axis.
  • center: Items are centered along the cross axis.
  • baseline: Items are aligned such that their baselines align.

8. Explain flex-wrap and its values.

Answer

The flex-wrap property specifies whether flex items are forced into a single line or can wrap onto multiple lines.

the values of flex-wrap:

  • nowrap (default): All flex items will be on one line.
  • wrap: Flex items will wrap onto multiple lines from top to bottom.
  • wrap-reverse: Flex items will wrap onto multiple lines from bottom to top.

9. What is the shorthand property for flex-direction and flex-wrap?

Answer

The shorthand property is flex-flow.

10. What is align-content and when is it used?

Answer

The CSS align-content property sets the distribution of space between and around content items along a flexbox's cross axis, or a grid or block-level element's block axis.

It's similar to justify-content but for the cross axis and only applies when there's wrapping. If there's only one line of flex items, align-content has no effect.

Common values include flex-start, flex-end, center, space-between, space-around, stretch (default).

11. Explain the flex shorthand property for flex items.

Answer

The flex shorthand property sets the flex-grow, flex-shrink, and flex-basis properties on a flex item.

  • flex-grow: Defines the ability for a flex item to grow if necessary (default is 0).
  • flex-shrink: Defines the ability for a flex item to shrink if necessary (default is 1).
  • flex-basis: Defines the default size of an element before any free space is distributed (default is auto).

Examples:

  • flex: 1; is equivalent to flex: 1 1 0%; (grow, shrink, and take up 0% basis, but grow to fill available space).
  • flex: auto; is equivalent to flex: 1 1 auto; (grow, shrink, and take up content size).
  • flex: none; is equivalent to flex: 0 0 auto; (do not grow, do not shrink, and take up content size).

12. What is flex-grow and when would you use it?

Answer

The flex-grow property specifies how much a flex item will grow relative to the rest of the flex items in the container if there is positive free space. A value of 1 means it will take up an equal share of the remaining space. A value of 2 means it will take up twice as much as an item with flex-grow: 1;.

It's used when you want certain items to expand and fill available space.

13. What is flex-shrink and when is it useful?

Answer

The flex-shrink CSS property sets the flex shrink factor of a flex item. If the size of all flex items is larger than the flex container, the flex items can shrink to fit according to their flex-shrink value. Each flex line's negative free space is distributed between the line's flex items that have a flex-shrink value greater than 0.

14. What is flex-basis?

Answer

The flex-basis property defines the initial size of a flex item before any free space is distributed. It can be a length (e.g., 200px) or a keyword (e.g., auto, which means the content's size).

It's essentially a more flexible way to set an item's width (or height if flex-direction is column) within a flex context, as it's considered before flex-grow or flex-shrink.

15. How do you center an item both horizontally and vertically using Flexbox?

Answer

To center an item both horizontally and vertically, you apply justify-content: center; and align-items: center; to the flex container.

16. What is the order property and how does it work?

Answer

The order property controls the order in which flex items appear in the flex container, overriding their source order. Items are laid out in ascending order of their order value. Items with the same order value are laid out in their source order.

The default value for order is 0. Negative values are allowed.

17. What is align-self and when would you use it?

Answer

The align-self property allows individual flex items to override the align-items property set on the flex container. It controls the alignment of a single flex item along the cross axis.

It accepts the same values as align-items (auto (default), flex-start, flex-end, center, baseline, stretch).

It's useful when you have a general alignment for most items but need to adjust one or a few items differently.

18. Can a flex item also be a flex container? Provide an example.

Answer

Yes, absolutely! This is a fundamental concept in creating complex layouts with Flexbox, often referred to as "nesting" flex containers. 

<div class="outer-flex-container" style="display:flex;"> 
    <div class="flex-item-1">Item1</div> 
    <div class="flex-item-2inner-flex-container" style="display: flex; flex-direction:column;"> 
        <div class="inner-item-A">Inner A</div> 
        <div class="inner-item-B">Inner B</div> 
    </div> 
    <div class="flex-item-3">Item 3</div> 
</div>

In this example, .outer-flex-container is a flex container. .flex-item-2 is a flex item within the outer container, but it is also an inner-flex-container, which in turn holds .inner-item-A and .inner-item-B as its own flex items.

19. What are the advantages of using Flexbox over traditional layout methods (floats, positioning)?

Answer

Advantages of Flexbox include:

  • Simplicity: Easier to create complex, dynamic layouts with less code.
  • Responsiveness: Excellent for responsive design as items can grow or shrink to fit the available space.
  • Alignment and Distribution: Powerful tools for aligning and distributing space among items along either axis (justify-content, align-items, align-content).
  • Order Control: Ability to visually reorder items using the order property without changing the HTML source.
  • Centering: Effortless horizontal and vertical centering.
  • No Floats or Clearfixes: Eliminates the need for floats and the associated clearfix hacks, leading to cleaner markup.
  • Gap Property: Provides a clean way to add spacing between flex items using the gap property (though it's a newer addition and not supported in all older browsers).

20. When would you choose CSS Grid over Flexbox, and vice-versa?

Answer
  • Choose Flexbox when: when You need to lay out items in a single dimension (either a row or a column).
  • Choose CSS Grid when: You need to lay out items in two dimensions (rows and columns) simultaneously. It's ideal for overall page layouts or complex sections where you want to explicitly define grid lines and areas for content.
